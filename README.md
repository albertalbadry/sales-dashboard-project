# 📊 Sales & Profit Dashboard

## 📌 Project Overview

This project focuses on analyzing sales and profit data to uncover meaningful business insights.
Using data analysis techniques and visualization, the project transforms raw sales data into an interactive dashboard that helps understand business performance and trends.

The analysis highlights key metrics such as total sales, profit performance, product categories, and regional performance.

---

## 🎯 Project Objectives

The main objectives of this project are:

* Analyze overall **sales and profit performance**
* Identify **top-performing product categories**
* Understand **monthly sales trends**
* Determine **high-performing states and regions**
* Discover **products generating the highest revenue**
* Provide an easy-to-read **visual dashboard for decision making**

---

## 🗂 Dataset Description

The dataset contains sales transaction data including:

| Column        | Description                     |
| ------------- | ------------------------------- |
| Order Date    | Date of the order               |
| Customer Name | Name of the customer            |
| State         | Customer location               |
| Category      | Product category                |
| Sub-Category  | Product sub-category            |
| Product Name  | Name of the product             |
| Sales         | Revenue generated               |
| Quantity      | Number of items sold            |
| Profit        | Profit made from the order      |
| Month         | Extracted month from order date |
| Year          | Extracted year from order date  |

## 📊 Dashboard Features

The dashboard provides visual insights including:
